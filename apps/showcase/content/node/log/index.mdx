# Log

Universal pretty and verbose logging

## Instalation

```bash
npm install @mvdlei/log
```

## Usage

Use it without any configuration.

```ts
import { logger } from "@mvdlei/log";

// call with any parameters and then watch your console.
logger.info;
logger.debug;
logger.log;
logger.error;
logger.warn;
```
